# 🤖 AI Token Analytics Dashboard

Дашборд для аналитики использования токенов AI-моделей.

## Возможности

- 📊 Загрузка и анализ CSV логов
- 💱 Конвертация в рубли по курсу ЦБ РФ
- 📈 Графики динамики затрат
- 🔑 Статистика по API-ключам
- 📋 Экспорт отчётов в HTML и CSV
